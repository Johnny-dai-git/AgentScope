════════════════════════════ Storage（模型存储）══════════════════════════════════

      ┌──────────────────────────────────────────────┐
      │ /model 挂载点（节点本地目录 → Pod 挂载）     │
      │ - 模型文件预先放在 GPU 节点本地磁盘         │
      │ - 通过 hostPath 等方式挂载到各 Worker Pod   │
      │ - 每个 Worker 容器内统一使用 /model 访问模型 │
      └──────────────────────────────────────────────┘


══════════════════════ NetworkPolicy（零信任网络边界）═══════════════════════════

   - 只允许 ingress-nginx → api-gateway-service
   - 只允许 llm-api Pods → vllm-worker-service
   - 默认拒绝跨命名空间访问
